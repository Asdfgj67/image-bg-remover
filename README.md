# 图片背景透明处理工具 🖼️→✨

一键将图片背景变为透明，保留主体人物/角色，输出指定尺寸的 PNG 图片。

## 功能

- 🎯 **AI + 传统算法**混合去背景，边缘精准
- 🖥️ **图形界面**，无需命令行，双击即用
- ⚙️ **可调参数**：输出尺寸、去背景力度、边缘平滑度、缩放比例
- 💾 **纯本地运行**，不上传、不联网、零隐私泄露
- 📦 **单文件夹分发**，无需安装 Python 或任何依赖

## 使用方式

### 方式一：直接运行（推荐）

下载 `dist` 文件夹，双击 `图片透明处理工具.exe`。

### 方式二：从源码运行

```bash
pip install pillow opencv-python numpy
python 图片透明处理工具.py
```

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 输出宽度 | 512 | 输出图片宽度 (px) |
| 输出高度 | 512 | 输出图片高度 (px) |
| 背景阈值 | 242 | 230~253，越高去背景越激进 |
| 边缘平滑 | 2 | 0=锐利边缘，5=柔和过渡 |
| 缩放比例 | 92% | 主体在画布中的占比 |

## 打包为 .exe

```bash
python build_exe.py
```

输出在 `dist/图片透明处理工具/` 文件夹。

## 技术栈

- Python + tkinter (GUI)
- OpenCV (GrabCut + 形态学处理)
- PIL/Pillow (图像 I/O 和缩放)
- NumPy (数据处理)
